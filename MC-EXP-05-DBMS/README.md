📘 MC Experiment 05 – DBMS (Android Studio)

📌 Aim

To develop an Android application that performs database operations like insert, update, and retrieve data using SQLite.

🧠 Theory

In Android, databases are managed using SQLite, which is a lightweight database system. It is used to store, retrieve, and manage data locally within the application.

Android provides built-in support for SQLite through classes like:

SQLiteDatabase

SQLiteOpenHelper

This helps in performing CRUD operations (Create, Read, Update, Delete) efficiently.

🛠️ Tools & Technologies

Android Studio

SQLite Database

Java / XML

Android Emulator / Physical Device

📂 Experiment Details

🔹 Database Creation

Database is created using SQLiteOpenHelper class inside the Android application.

🔹 Table Creation

Tables are created using SQL queries in the app.

🔹 Insert Data

User inputs are stored into the database.

🔹 Retrieve Data

Stored data is fetched and displayed on the screen.

📊 Output

Successfully created Android application

Data inserted into SQLite database

Data retrieved and displayed in the app UI

📌 Conclusion

This experiment helped in understanding how to integrate a database with an Android application and perform basic CRUD operations using SQLite.
